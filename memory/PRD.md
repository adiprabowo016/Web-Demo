# Portfolio Website - Abdullah Adi Prabowo

## Original Problem Statement
Build a web demo portfolio/CV website from GitHub repository (https://github.com/adiprabowo016/newweb). User requested dark theme design for their portfolio and CV website.

## Architecture
- **Frontend**: React 18 with React Router for SPA navigation
- **Backend**: FastAPI (minimal API for health check)
- **Styling**: Custom CSS with dark theme, CSS variables, responsive design
- **Fonts**: Space Mono (monospace), Outfit (sans-serif)

## User Personas
- **Primary**: Abdullah Adi Prabowo - Software Quality Assurance Engineer seeking job opportunities
- **Visitors**: Recruiters, potential employers, collaborators in tech industry

## Core Requirements (Static)
1. ✅ Dark theme design
2. ✅ Professional portfolio/CV presentation
3. ✅ Mobile responsive
4. ✅ Multiple pages (Home, Experience, Portfolio, Contact)
5. ✅ Contact form
6. ✅ Social media links
7. ✅ Download CV functionality

## What's Been Implemented
### 2024-02-09
- Created React-based portfolio from original HTML
- **Home Page**: Hero section with animated badge, name, title, description, stats (5+ years, 4 companies, 20+ projects)
- **Experience Page**: Timeline layout showing work history at Bank Mandiri, Asuransi Astra, Bank Indonesia, Gameloft
- **Portfolio Page**: Project cards showcasing QA testing and web development work
- **Contact Page**: Contact form, email/phone/location info, social links (LinkedIn, Instagram, GitHub)
- **Navigation**: Fixed navbar with smooth transitions, mobile hamburger menu
- **Footer**: Brand logo, social links, copyright
- **Design**: Dark theme with cyan (#00d9ff) and purple (#7c3aed) accents, glow effects, grid overlay
